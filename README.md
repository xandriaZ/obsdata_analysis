# obsdata_analysis

This repository contains coursework and data analysis notebooks for the **Observational Experiments of Astrophysics** course (AY2025/26 Semester 1).

## Overview

The repository consists of three Jupyter notebooks covering fundamental observational astronomy data reduction and analysis techniques, including CCD calibration, photometry, flux calibration, and stellar population studies.

## Contents

### 1. CCD Calibration

**Notebook:** `1_calibration.ipynb`

This notebook demonstrates the basic CCD data reduction workflow:

* Bias correction
* Dark correction
* Flat-field calibration
* Creation of master calibration frames
* Application of calibration frames to science images

### 2. M33 Flux Calibration and Photometry

**Notebook:** `M33_fluxcalib_phot.ipynb`

Analysis of observations of the Triangulum Galaxy (M33), including:

* Aperture photometry
* Flux calibration
* Magnitude determination
* Conversion between instrumental and physical flux measurements

### 3. M15 Hertzsprung–Russell Diagram

**Notebook:** `M15_HRdiagram.ipynb`

Construction and analysis of the Hertzsprung–Russell (HR) diagram for the globular cluster M15:

* Photometric data processing
* Color–magnitude diagram generation
* Stellar population analysis
* Identification of evolutionary sequences
